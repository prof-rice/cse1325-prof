The Makefile in this directory does NOT make the files in this directory.

Instead, it may be copied to your own full_credit directory to build
your Purse class with both your bank.cpp and the test_full_credit.cpp
main functions.

make will build your purse.h, purse.cpp, and main.cpp executable as bank.

make test_full_credit will build your purse.h and purse.cpp along with
test_full_credit.cpp as executable test_full_credit, a regression test
that prints nothing when it succeeds. Use the -v command line argument
if you'd like to see each test's name as it runs.

Copy Makefile_bonus to your bonus directory and rename it Makefile.

Copy Makefile_extreme_bonus to your extreme_bonus directory and
rename it Makefile.
